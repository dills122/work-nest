# Work Nest Task Management App (Frontend)

This is the frontend for the Work Nest app for creating & managing tasks/to-do lists.

## Setup

You'll need `Node` to run this app.

```bash
# configure correct Node Version
nvm use lts/iron
# install everything
npm i

# Start FE App
npm run dev
# Or
npm run build
npm run start
```
